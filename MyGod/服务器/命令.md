# docker创建mysql-Percona镜像


    ```
    docker run --name mysql -p 9606:3306 --restart always --privileged=true -v /usr/mygod/mysql/data:/var/lib/mysql -e MYSQL_ROOT_PASSWORD=mysql -d percona:5.7.26 --character-set-server=utf8 --collation-server=utf8_general_ci

	   --name                                   ：服务名
	   -p                                       ：端口映射   主机:容器
	   --restart always                         ：开机启动
	   --privileged=true                        ：提升容器内权限
	   -v /usr/mygod/mysql/data:/var/lib/mysql  ：设置自定义数据目录
	   -e MYSQL_ROOT_PASSWORD=mysql             ：设置参数，root密码
	   -d                                       ：设置后台运行
	   percona:5.7.26                           ：运行的镜像文件
	   --character-set-server=utf8              ：设置数据编码集
	   --collation-server=utf8_general_ci       ：排序规则

   	注意：上面命令第一次执行时有可能不会成功，具体原因未知，后来是通过使用mysql镜像创建成功后在使用上面命令创建成功，有可能是目录权限引起
    ```

    docker run --name redis -p 9679:6379 --restart always  -v /usr/mygod/redis/data:/data -d redis:5.0.6 --requirepass "mypassword"
    requirepass  ： 设置密码
